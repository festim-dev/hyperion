# hyperion
- **t1.py** — Dry-run simulations at 500 °C–700 °C, testing both lower and upper permeability limits.  
- **t3.py** — Transient simulation for FLiBe permeability with adaptive time-stepping enabled.  
- **t4.py** — Steady-state FLiBe permeability case with three different boundary conditions and corresponding outputs.  
- **t5.py** — Input file for the flow-direction swap test.
